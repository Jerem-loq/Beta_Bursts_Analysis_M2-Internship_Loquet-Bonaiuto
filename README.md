## Overview

This repository contains the notebook-based analysis for regular and generalized contrastive PCA (`rPCA` and `gcPCA`) applied to STN and sensorimotor cortex beta burst data. The primary goals are:

- compare group-level gcPCA and rPCA performance
- evaluate classification accuracy as a function of percentiles and principal components
- inspect burst count and mean waveform patterns across brain regions
- reproduce notebook workflows and generated outputs

The dataset and methodological context are based on Agouram et al. (2025): https://doi.org/10.1101/2025.07.09.25331208.

## Notebook navigation

### Tutorial

- `gcpca_tutorial1.ipynb`
  - Toy dataset demonstration of PCA vs gcPCA
  - shows gcPCA outputs, interpretation, and comparison with standard PCA
  - de Oliveira et al., 2025 - 10.1371/journal.pcbi.1012747

### Core group-level workflows

- `Group-Level_rPCA_vs_gcPCA-Acc_AsFunctionOf_Percentiles.ipynb`
  - compares gcPCA and rPCA classification accuracy across percentile-based binning
  - this notebook provides the main group-level benchmark comparison

- `Group-Level_gcPCA_Accuracy_AsFunctionOf_gcPCs.ipynb`
  - evaluates gcPCA performance as a function of the number of gcPCs
  - includes cross-validation and fold-level score reporting

- `Group-Level_gcPCA-Accuracy_AsFunctionOf_percentiles.ipynb`
  - plots gcPCA logit accuracy across percentile binning choices
  - includes cross-validation and fold-level score reporting

- `Group-Level_gcPCA_Burst-Count_MeanWaveforms.ipynb`
  - analyzes gcPCA-derived and rPCA-derived waveform bins and burst counts
  - compares mean burst waveforms and count statistics across train/test splits

- `Group-Level_rPCA_MeanWaveforms.ipynb`
  - explores mean waveforms for rPCA bins and group-level waveform structure

- `Group-Level_Accuracy_AsFunctionOf_TFA.ipynb`
  - inspects time-frequency auditing features for classifier performance
  - compares burst TF feature distributions across subjects and brain regions

- `Group-Level_From-PCA-to-Waveforms.ipynb`
  - traces the PCA transformation from raw burst waveforms to classification features

- `Group-Level_Results_Comparison_plots.ipynb`
  - summarizes the best models across brain regions, PC counts, and binning
  - visualizes coefficient trends and selected gcPCA models

### Subject-level workflows

- Same logic as previously described with the Group-Level analyses
